# Geshem

An interactive rain radar clone running on Mapbox GL, see it live at: [https://geshem.space](https://geshem.space)

## Dev

 - Node + Yarn
 - Redis (optional, recommended)

Install dependencies:
```bash
$ yarn install
```

Run a local cache:
```bash
$ redis-server
```

Run a webpack watch process:
```bash
$ webpack --watch
```

Run the server:
```bash
$ node server.js
```
